# CNN-LeNet5

Un projet implémentant le modèle **LeNet-5**, un réseau de neurones convolutionnel classique conçu pour la reconnaissance de caractères manuscrits (MNIST). Ce projet offre une implémentation du modèle LeNet-5 en Python, permettant l’entraînement, la validation et l’évaluation sur le dataset MNIST.

## 📋 Description

LeNet-5 a été introduit par **Yann LeCun** en 1998 et constitue l'un des premiers réseaux convolutionnels à succès. Il est spécifiquement conçu pour la classification des chiffres manuscrits, rendant son architecture simple et efficace pour des tâches similaires.

Ce projet implémente les étapes suivantes :
- Chargement du dataset MNIST.
- Construction et entraînement du modèle LeNet-5.
- Évaluation des performances du modèle.

## 🚀 Fonctionnalités

- **Architecture LeNet-5 complète** :
  - 2 couches convolutionnelles.
  - 2 couches sous-échantillonnage (max pooling).
  - 3 couches entièrement connectées.
- **Support de MNIST** : Chargement et prétraitement automatique des données.
- **Évaluation du modèle** : Calcul des métriques de performance, comme la précision.
